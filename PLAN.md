- 사용자가 입력한 텍스트 유실 없이 모두 텍스트화되도록 고도화 (AI모델이 임의로 걷어내는 부분 최소화하도록)
- 페이지 관리 체계 고도화 (노션처럼 사용자가 문서들 관리할 수 있도록)
- 사용자가 직접 해당 문법으로 수정가능한 기능 추가 (마크다운 툴도 제공. 해당 문법으로 굳이 적지 않아도 텍스트 툴들을 제공해 직접 텍스트 디자인 가능하도록)
- 사용자 커스텀 프롬프트 설정 (간결모드, 설명모드 등 모드 고를 수 있게), 자신의 말투, 형식, 내용 그대로 유지하고플수도 있으니 텍스트 변동률 지정 (0~100%)
- 디렉토리 형식, 디자인 자신이 커스텀 가능하게 (메모지형, 마인드맵형, 복합)
- 사용자 커스텀 프롬프트 - 사용자 지정 규칙 세팅 기능 (ex) 모든 메모에는 레퍼런스
- 사용자 커스텀 프롬프트 심화 - 형식 전부 예제로 시각화해서 선택 가능하게. 깃허브 컨플릭트 화면처럼. 간결vs복잡버전은 택1, 기타 내부 디자인은 한 블럭씩 깃허브 머지 코드라인처럼 고를수있게
- 스트리밍 응답으로 변환 중 실시간 결과 표시
- 포럼 생성 (부트스트랩처럼 자신의 문서화 디자인 공유하는)
- 가상유저들 포럼에 공유디자인 생성
- 노션, 기타 유명한 문서화 툴로 내보내기 기능
- 사용자 활동,접속 로깅, 페이지 구축
- 관리자용 페이지 구축
- 마케팅 - 인스타용 테스트 사이트 생성(mbti테스트 같은 2~4지선다식)
- 수익화 - 초기에는 얼리버드세일넣기
- 서버, DB 구축

- 구독형 결제 시스템 구축
